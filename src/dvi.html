Калькулятор 2.0. Инструкция по применению.

Арифметические операторы:

  • Сложение : a + b
  • Вычитание : a - b
  • Скобки : (a + b)
  • Умножение : a * b
  • Деление : a / b
  • Возведение в степень : a ^ b
  • Остаток от деления : a mod b
  • Унарный плюс: +a
  • Унарный минус : -a

Функции:

  • Вычисляет косинус : cos(x)
  • Вычисляет синус : sin(x)
  • Вычисляет тангенс : tan(x)
  • Вычисляет арккосинус : acos(x)
  • Вычисляет арксинус : asin(x)
  • Вычисляет арктангенс : atan(x)
  • Вычисляет квадратный корень : sqrt(x)
  • Вычисляет натуральный логарифм : ln(x)
  • Вычисляет десятичный логарифм : log(x)

Также калькулятор имеет кнопку "x", которая записывает 'x' в выражение. Вы можете посчитать выражение с 'x', введя значение в поле ввода, и калькулятор посчитает выражение с заданным значением "x". Имеется возможность записи числа в экспоненциальном виде с помощью кнопки "e"

Если вы хотите отчистить поле ввода, то нажмите кнопку "AC"

В калькуляторе можно посроить график заданной функции с возможностью выбора масштаба. Для этого введите функцию и нажмите кнопку "PLOT".

В калькуляторе предусмотрен режим кредитного калькулятора. Чтобы перейти в данный режим, нажмите кнопку "Credit Calc" в верхней части окна. Вход: общая сумма кредита, срок, процентная ставка, тип (аннуитетный, дифференцированный) Выход: ежемесячный платеж, переплата по кредиту, общая выплата.
